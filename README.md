# Tripleten web_project_homeland

Proyecto #5 para el bootcamp de tripleten en desarrollo web.

John Diaz Ayala

link:https://jdj-98.github.io/web_project_homeland/

En este proyecto se abordan los conceptos de diseño responsivo, dando lugar a la adpatación de la pagina en cuestión a 3 diferentes resoluciones que abarcan algunos de los dispositivos del mercado. Se hizo uso de la metodología BEM tanto en la estructura del proyecto como en el diseño en si de la pagina. Se hizo uso de diseño responsivo por medio de media queries para que la pagina se adapte a las distintas resoluciones definidas en el brief y también se aplicaron tecnicas de diseño para la tipografía.
